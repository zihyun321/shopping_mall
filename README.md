# 쇼핑몰 구축 프로젝트

`사용언어`
Server: Node Express
Front: React
CSS: Tailwind
DB: MySQL

### 1. 설치

`DB 구축`
MySQL 다운로드: https://www.mysql.com/ 

`React 설치`
npx install create-react-app [프로젝트명]

`Node 설치`
npm init
npm install express

`React&Node 동시 실행`
npm i -D concurrently

`DB 설치`
npm install --save mysql2
